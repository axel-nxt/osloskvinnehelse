# SEO Dashboard — OsloKvinnehelse

Et premium SEO-dashboard bygget med React + Vite for OsloKvinnehelse-presentasjon.

## Kom i gang

```bash
npm install
npm run dev
```

## Seksjoner

- **Domenestrategi** — Interaktiv Alt A / Alt B velger for domenestrategi
- **Teknisk Analyse** — Screaming Frog-funn forklart på norsk
- **SEO & Synlighet** — Nøkkelordrangeringer og potensial
- **Konkurrentanalyse** — VS Oslo Kvinnehelsesenter
- **Tiltaksliste** — Prioritert handlingsplan
